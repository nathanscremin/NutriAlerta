# Definition of Done (DoD) & Testes

Para que uma tarefa seja considerada "Concluída", você deve garantir:

1. **Validação de Dados:** O script deve rodar com dados reais (não mock) e gerar logs de erro claros.
2. **Documentação:** README atualizado e comentários em inglês explicando a lógica do Random Forest.
3. **UI/UX:** Se houver mapa, as cores devem seguir o `style-guide.md` (Ex: Vermelho para alto risco).
4. **Revisão:** O código deve estar pronto para o Pull Request (coluna 'Em Revisão' no Trello).
